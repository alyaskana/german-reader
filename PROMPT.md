# Как сгенерировать новую историю

Приложение само собирает персональный промпт: на главном экране нажми **«Скопировать промпт»** — в него автоматически подставятся твои оценки сложности прочитанных историй, слова, которые ты учишь, и слова, которые тебе уже знакомы.

Дальше:

1. Вставь промпт в Claude ([claude.ai](https://claude.ai)).
2. Получи в ответ JSON истории.
3. Сохрани его как `src/stories/<id>.json`.
4. Добавь импорт в `src/stories/index.ts`:

```ts
import meineNeueGeschichte from './meine-neue-geschichte.json'

export const stories: Story[] = [
  // ...существующие,
  meineNeueGeschichte,
]
```

5. Закоммить и запушь — GitHub Pages пересоберётся автоматически.

## Формат истории

```json
{
  "id": "kebab-case-id",
  "title": "Заголовок на немецком",
  "titleRu": "Перевод заголовка",
  "level": "A1",
  "paragraphs": [
    "Текст абзаца. Сложные слова размечаются так: {{Wort|перевод}}.",
    "Ещё абзац..."
  ],
  "dict": { "слово": "перевод в контексте" },
  "cover": "<svg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 200 120' preserveAspectRatio='xMidYMid slice'>...</svg>"
}
```

Правила разметки:
- `{{Wort|перевод}}` — слово или выражение целиком ({{zu Hause vergessen|забыл дома}}) в той форме, в которой стоит в тексте; перевод короткий, на русском.
- Разорванные единицы (отделяемая приставка, Perfekt): позднюю часть помечай `{{+auf}}` — «Er {{steht|встаёт (aufstehen)}} früh {{+auf}}.» Перевод только у первой части, с инфинитивом.
- Глоссируй каждую единицу только при первом появлении.
- 8–12 глосс на историю, 120–180 слов, 4–6 абзацев.
- `dict` — переводы всех остальных слов текста (ключи в нижнем регистре, без имён собственных), по смыслу в контексте истории; части составных форм объясняются с отсылкой к целому.
- `cover` — обложка: inline SVG одной строкой, плоская иллюстрация по сюжету без текста и людей. `viewBox='0 0 200 120'`, атрибуты в одинарных кавычках, только простые фигуры (`rect`, `circle`, `ellipse`, `path`), палитра приложения (фон `#f6efe4`, акцент `#d9534f`, охра `#e8c07d`, шалфей `#a8b894`, синий `#7d95a8`). Перед показом SVG проходит проверку (без `script`, обработчиков событий и внешних ссылок); если поля нет или оно не прошло проверку — рисуется нейтральная обложка-заглушка.

Тап по любому слову текста показывает перевод: сначала ищется глосса, потом `dict`; если слова нет нигде — приложение предложит внешний переводчик.
