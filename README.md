# 📖 Lesezeit — истории на немецком (A1)

Минималистичный ридер адаптированных историй на немецком, как в книгах для изучения языка: сложные слова выделены, над ними — мелкий перевод на русский.

## Что умеет

- **Чтение с глоссами** — переводы у сложных слов и выражений (в том числе разорванных: отделяемые приставки, Perfekt); режим «по тапу» для самопроверки.
- **Перевод любого слова** — у каждой истории есть словарь всех её слов с контекстными переводами; тапни любое слово текста, даже не выделенное.
- **Мои слова** — тапни слово → «+ В мои слова», собирай личный словарик.
- **Тренировка слов** — флешкарточки без расписания: листай, когда захочешь. «Уже знаю» — и слово перестаёт подсвечиваться в текстах (в словарике можно вернуть в изучение).
- **Обложки** — плоские SVG-иллюстрации у историй: в списке карточкой, на странице — под заголовком. Генерируются вместе с историей (Claude рисует SVG кодом).
- **Оценка сложности** — после истории: легко / норм / сложно.
- **Свои истории без правки кода** — экран «Добавить историю»: скопируй персональный промпт (в нём твои оценки, слова, которые учишь, и уже выученные), сгенерируй историю в любом привычном ИИ-чате (Claude, ChatGPT, Gemini…) и вставь JSON прямо в приложение. Подробнее в [PROMPT.md](PROMPT.md).

Всё хранится в localStorage, бэкенда нет.

## Запуск

```bash
npm install
npm run dev
```

## Деплой на GitHub Pages

При пуше в `main` workflow `.github/workflows/deploy.yml` собирает и публикует приложение. В настройках репозитория включи **Settings → Pages → Source: GitHub Actions** (один раз).

## История дня (автогенерация)

Каждый день короткая история, построенная вокруг слова или идиомы дня (с учётом сезона и праздников), генерируется автоматически и коммитится в `src/stories/daily-YYYY-MM-DD.json`. Пуш запускает деплой — история сама появляется на сайте, в сборнике «История дня» и карточкой на главной.

Генерация настроена через **Routine (запланированный триггер) Claude Code**: раз в день просыпается сессия Claude Code, пишет историю по заданной инструкции, коммитит и пушит в `main`. Отдельный API-ключ не нужен — генерация идёт через подписку Claude Code, а не через оплату API.

Истории общие для всех (персонализация под твои слова/оценки остаётся в браузере). Формат каждой истории — как в существующих `src/stories/daily-*.json`.

## Перенос в отдельный репозиторий

Этот код лежит в ветке профильного репо. Чтобы вынести в свой репо `german-reader`:

```bash
# 1. Создай пустой репо german-reader на github.com (без README)
# 2. Затем:
git clone -b claude/german-a1-reader-spa-lujca5 https://github.com/alyaskana/alyaskana.git german-reader
cd german-reader
git remote set-url origin https://github.com/alyaskana/german-reader.git
git checkout -b main
git push -u origin main
```

После пуша включи Pages (Source: GitHub Actions) — приложение появится на `https://alyaskana.github.io/german-reader/`.
